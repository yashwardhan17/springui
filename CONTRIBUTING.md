# Contributing to SpringUI

SpringUI is in early alpha. We need contributors at every level.

## Good first issues
- Writing documentation
- Building the example todo-app
- Implementing VNode (the Virtual DOM node class)
- Writing unit tests for the diff engine

## How to contribute
1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Open a pull request with a clear description

## Discussion
Open a GitHub Discussion for ideas, questions, or architecture proposals.
